What does HTML stand for?

HTML stands for HyperText Markup Language. It is the standard markup language for creating web pages.

What is the purpose of the <head> section in an HTML document?

The <head> section contains metadata about the document, such as the title, links to stylesheets, and scripts. It is not displayed directly on the page.

How do you create a hyperlink in HTML

You can create a hyperlink using the <a> tag. For example: <a href="https://www.example.com">Visit Example</a>.

What does CSS stand for, and what is its purpose

CSS stands for Cascading Style Sheets. It is used to style HTML elements, controlling layout, colors, fonts, and overall presentation of a web page.

How can you apply CSS to an HTML document

CSS can be applied in three ways: inline styles using the style attribute, internal styles within a <style> tag in the <head>, or external styles using a linked .css file with the <link> tag.

What is the box model in CSS?

The box model describes the structure of elements on a web page, consisting of margins, borders, padding, and the content area. It determines how elements are sized and spaced.

What is JavaScript, and how is it used?

JavaScript is a programming language used to create interactive and dynamic content on web pages. It allows developers to manipulate HTML and CSS, respond to user events, and handle data

How do you declare a variable in JavaScript?

You can declare a variable using var, let, or const. For example: let name = "John";.

What is the difference between == and === in JavaScript?

== checks for value equality with type coercion, while === checks for both value and type equality (strict equality).

What is the Document Object Model (DOM)?

The DOM is a programming interface that represents the structure of a web document. It allows JavaScript to interact with HTML and XML documents, enabling dynamic changes to the content and structure of web pages.
